A Time Picker for Ionic Framework
Click on the below button(s) to open time picker popup and set the time.
Then you can see that the 'epoch time' will change as per the selected time.

12-Hour Format

Epoch time is : {{slots[0].epochTime}} seconds.

{{slots[0].step}} minute increment.

24-Hour Format
