##Introduction:

This is a sample application to demonstrate `ionic-timepicker` bower component which can be used with any Ionic framework's application.

[View Demo](http://rajeshwarpatlolla.github.io/TimePickerForIonicFramework/demo/ "Demo") 

Please check the [ionic-timepicker](https://github.com/rajeshwarpatlolla/ionic-timepicker) bower component for more details.
